# tic-tac-toe-multiplayer

Multiplayer Tic Tac Toe web game

Try it here: https://tic-tac-toe227.netlify.app/

## Technologies used

    ReactJs
    NodeJs
    Socket io

## How to run

    Start server:
        Go to server folder
        run command "npm install"
        run command "npm start"
    Start client:
        Go to client folder
        run command "npm install"
        run command "npm start"
